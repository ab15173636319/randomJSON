# RandomJSON 应用说明

## 项目概述
RandomJSON 是一个基于 Vue 3 和 Vite 构建的**随机虚拟数据生成工具**，能够根据用户定义的字段结构批量生成随机JSON数据，并自动复制到剪贴板。

## 核心功能

### 📝 字段配置
- **多种数据类型**：支持 `string`、`number`、`float`、`boolean`、`Date`、`中文`、`ID`、`Image`、`Avatar`
- **自定义参数**：可根据不同类型设置长度、尺寸等参数
- **表单验证**：自动验证参数名称格式（字母、数字、下划线）和必填项
- **可视化管理**：通过表格形式查看和编辑字段配置
- **自动重置**：添加字段后自动清空表单，方便连续添加

### 🔧 数据生成
- **批量生成**：可设置生成数据条数（默认10条）
- **自动复制**：生成后自动复制JSON数据到剪贴板
- **数据预览**：生成的数据以代码块形式显示在页面中
- **错误提示**：当未配置字段时，会提示错误信息

### 🛠️ 操作功能
- **添加字段**：通过表单添加新的数据字段（带验证）
- **修改字段**：抽屉式界面修改字段属性
- **删除字段**：快速移除不需要的字段
- **撤销功能**：一键清空所有字段配置

## 使用流程

### 1. 添加数据字段
在顶部"添加新参数"区域配置：
- 输入参数名称（支持字母、数字、下划线，1-16个字符）
- 选择数据类型（必填）
- 根据类型设置相应参数（如长度、尺寸等）
- 点击"添加参数"（会自动验证并重置表单）

### 2. 管理字段配置
在表格区域可以：
- 查看当前所有字段配置（属性名、数据类型、长度等）
- 点击"修改"按钮调整字段属性（抽屉式界面）
- 点击"删除"按钮移除字段
- 操作按钮采用网格布局，方便快速操作

### 3. 生成JSON数据
- 在底部面板设置"数据长度"（生成多少条记录，默认10条）
- 点击"生成代码"按钮
- 生成的数据会以代码块形式显示在页面中，并自动复制到剪贴板
- 如果未配置任何字段，会显示错误提示

### 4. 撤销配置
- 点击底部面板的"撤销"按钮
- 一键清空所有已配置的字段
- 方便重新开始配置

## 数据类型详解

| 类型      | 特性         | 可设置参数 |
| --------- | ------------ | ---------- |
| `string`  | 随机字符串   | 长度       |
| `number`  | 随机整数     | 最大位数   |
| `float`   | 随机浮点数   | 数值范围   |
| `boolean` | 随机布尔值   | 无         |
| `Date`    | ISO日期格式  | 无         |
| `中文`    | 随机中文姓名 | 字符数量   |
| `ID`      | UUID格式ID   | 无         |
| `Image`   | 随机图片URL  | 宽度、高度 |
| `Avatar`  | 随机头像URL  | 尺寸       |

## 技术特点

### 🎨 界面设计
- **护眼配色**：采用米色、浅黄、浅绿等柔和色调，降低视觉疲劳
- **响应式布局**：完美支持桌面端和移动端访问
- **毛玻璃效果**：现代化半透明背景和模糊效果
- **柔和阴影**：使用低对比度阴影，更加舒适
- **代码高亮**：深绿色背景配合浅绿色文字，护眼且清晰
- **优化交互**：按钮悬停效果、表格行高亮、平滑过渡动画

### ⚡ 性能优化
- 使用 Pinia 进行状态管理，响应式数据更新
- 深拷贝功能避免数据污染
- 丰富的工具函数支持（ID生成、日期生成、随机字符串等）
- 表单验证优化，提升用户体验
- 自动重置表单，提高操作效率

### 📱 设备兼容
- 支持现代浏览器（Chrome、Firefox、Safari、Edge等）
- 移动端完美适配（响应式布局、触摸优化）
- 提供Vue DevTools调试支持
- 自适应字体大小和间距

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 使用场景

这个工具特别适用于：
- **前端开发**：快速生成Mock数据用于组件开发和测试
- **API测试**：生成符合接口要求的测试数据
- **组件演示**：为UI组件提供随机数据展示
- **原型开发**：快速搭建需要数据的原型界面
- **学习练习**：理解JSON数据结构和随机数据生成

## 注意事项

- 参数名称必须符合规范（字母、数字、下划线，1-16个字符）
- 数据类型为必选项，添加前请确保已选择
- 生成数据前请确保至少配置了一个字段
- 撤销操作会清空所有配置，请谨慎使用
- 生成的数据会自动复制到剪贴板，可直接粘贴使用
